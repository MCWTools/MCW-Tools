How to use
1.Download python
https://www.python.org/downloads/
2.Download cryptography
pip install cryptography
3.Run file
python (or py on 32-bit device) Encrypt.py
4.Get file path
Left click the file, press Ctrl + Shift + C
5.Paste in
Ctrl + V
6.Copy the secret key
If you want to decrypt just Ctrl Shift C the encrypted file (File has extension is .jpgenc , txtenc, jsonenc, pngenc,...)
Then paste in the secret key and youre done!
